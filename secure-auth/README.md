# SecureAuth

A secure, sandboxed authentication web application built with Python and Flask.
Designed for OS-level academic projects focusing on core security, buffer overflow protection, anti-brute-force, and Email OTP Two-Factor Authentication.

---

## Prerequisites

**Python 3.10 or higher** must be installed.

- Download from: https://www.python.org/downloads/
- During installation, check **"Add Python to PATH"**
- Verify: open a terminal and run `python --version`

---

## Quick Start (Windows)

### Step 1 — Clean existing data (If migrating from older version)
Delete the `instance/auth.db` or `auth.db` file to start fresh.

### Step 2 — Run setup (one time only)

Double-click `setup.bat`, or run in a terminal:

```
setup.bat
```

This will:
1. Create a `venv\` virtual environment
2. Install all Python packages into it
3. Generate a `.env` file with a fresh secret key

### Step 3 — Start the app

```
venv\Scripts\activate
python run.py
```

### Step 4 — Open in browser

```
http://127.0.0.1:5000
```

## Authentication Flow

```
REGISTRATION
  Fill form → OTP sent to email → Enter OTP → Account created → Logged in ✓

LOGIN
  Enter email + password → OTP sent to email → Enter OTP → Logged in ✓

FORGOT PASSWORD
  Enter email → OTP sent → Enter OTP + new password → Logged in ✓
```

## Security Features

| Feature | Implementation |
|---------|---------------|
| Multi-Factor Auth (MFA) | Email OTP using Gmail SMTP |
| Password policy | 12+ chars, upper/lower/digit/symbol |
| Password hashing | bcrypt (adaptive cost factor 12) |
| CSRF | Flask-WTF hidden token on every form |
| Buffer Overflow & Injection | SQLAlchemy ORM + strict input length/format validation |
| Brute force | 5 attempts → 15-min lockout; rate limit 10/min per IP |
| Timing attacks | Dummy bcrypt hash on unknown email; `hmac.compare_digest` for OTP |
| Session security | HttpOnly + SameSite=Lax cookies |
| Privilege misuse | @role_required decorator; role modifications protected |
| Audit trail | Every security event logged with timestamp + IP |

## Resetting the App

To start completely fresh, just delete:

```
instance/auth.db  ← wipes all users
venv\             ← wipes all packages
.env              ← wipes secrets (re-generated by setup.bat)
```
